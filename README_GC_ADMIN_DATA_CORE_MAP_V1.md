# GC Admin Data Core Map v1

Documento de separación de dominios para GrassCutters WebNode.

Resumen:

- Race Data Core = Stracker
- Championship Core = ACSM / campeonato
- Archive Core = archivo motorsport / media
- Identity/Profile Core = usuarios / pilotos vinculados
- Calendar Core = calendario manual/publicable

ACSM no debe alimentar /api/gc/active-combo. ACSM queda para la página del campeonato.
