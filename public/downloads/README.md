# Descargas públicas GrassCutters

Coloca aquí el APK compilado de Android con este nombre recomendado:

```txt
grasscutters-android.apk
```

La página `/app-android` apunta por defecto a:

```txt
/downloads/grasscutters-android.apk
```

Si prefieres alojar el APK fuera del proyecto, configura:

```env
PUBLIC_ANDROID_APK_URL=https://tu-dominio.com/ruta/grasscutters-android.apk
```
